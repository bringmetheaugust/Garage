# Garage🏚 server side

## Description

Server side for *@Garage* project.

## Deploy(local)

  * `yarn` install all dependencies
  * `yarn dev` run development server on **localhost:82**

  ##### Required local packages:

  * *yarn* v1+
  * *NodeJS* v14.15.3
